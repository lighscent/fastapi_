# Tutoriels et Ressources FastAPI

Voici une sélection structurée de vidéos pédagogiques pour t’accompagner étape par étape dans l'apprentissage de FastAPI :

## 🧰 1. Mise en place et installation
**Installation des outils - Tuto 1 FastAPI 2022**
Guide dans l’installation de FastAPI, Uvicorn, et la configuration de ton environnement de développement. Indispensable pour partir sur de bonnes bases.

## 👋 2. Premier projet : Hello World
**Hello World avec FastAPI - Tuto 3 FastAPI** ([Lien YouTube](https://www.youtube.com/watch?v=NhzqPSvT4A8))
Créer ta première API : définir une route, renvoyer une réponse JSON, et tester ton serveur local.

## 🧠 3. Comprendre les bases des routes et des requêtes
**Fast API : Créer facilement une API en Python**
Explore les types de requêtes (GET, POST, PUT, DELETE), la gestion des paramètres, et même une mini base de données.

## 🧪 4. Créer une API complète en 10 minutes
**Créez une API Python en 10 minutes avec FastAPI !** ([Lien YouTube](https://www.youtube.com/watch?v=0-yncL0bqZs))
Créer une API d’analyse simple, avec documentation interactive et tests.

## 🧬 5. Aller plus loin avec le machine learning
**Créer un API Python avec FastAPI pour ton modèle de ML** ([Lien YouTube](https://www.youtube.com/watch?v=7D_0JTeaKWg))
Montre comment exposer un modèle de deep learning via FastAPI, avec des exemples concrets.

## 🐳 6. Déploiement avec Docker
**Créer une API de machine learning avec FastAPI et Docker**
Dockeriser ton API pour la rendre portable et prête pour le cloud (avec notions de Kubernetes).

## 🧱 7. Formation complète en 2H
**Créer des API en Python avec FastAPI en seulement 2H**
Couvre la documentation, les exceptions, les tests, et même la création de tables SQL.

## 🧹 Commandes utiles

**Créer un environnement virtuel :**
```bash
python -m venv .venv
```

**Désinstaller toutes les libs pip d'un coup :**
```bash
pip freeze | xargs pip uninstall -y
```
